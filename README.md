# FPGA VGA Oscilloscope

## Overview

This project implements a real-time FPGA-based oscilloscope using Verilog HDL on a Xilinx Artix-7 FPGA. Analog signals are sampled through an ADC interface, stored using circular memory buffering, and rendered on a VGA display for real-time waveform visualization.

## Features

* Real-time waveform acquisition and display
* VGA output with custom HSYNC and VSYNC generation
* ADC integration for analog signal sampling
* Circular memory buffering for continuous waveform updates
* Support for square and triangle waveform visualization
* Hardware implementation on a Xilinx Artix-7 FPGA

## Technologies

* Verilog HDL
* Xilinx Vivado
* Xilinx Artix-7 FPGA
* VGA Interface
* ADC Interface
* Digital Logic Design
